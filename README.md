## Como usar

  ```shell
  1- inicialize o servidor mongoDB
  2- cd "seu diretório desejado"
  3- git clone https://github.com/JoseLucasapp/Reporte-desafio-tecnico
  4- cd Reporte-desafio-tecnico
  5- npm start
  ```
